# PDF 編輯器 📄✨

一個功能強大、美觀現代的 PDF 編輯器,支援網頁版和桌面版,提供完整的編輯、轉換、匯入匯出功能。

## ✨ 主要特色

### 🎨 現代化設計
- 精美的漸變色彩方案
- 深色/淺色主題切換
- 流暢的動畫效果
- 響應式佈局設計

### 🌍 多語言支援
- 🇹🇼 繁體中文
- 🇨🇳 簡體中文
- 🇬🇧 English
- 🇻🇳 Tiếng Việt

### 📝 完整編輯功能
- ✅ PDF 檔案上傳與預覽
- ✅ 頁面導航與縮放控制
- ✅ 文字工具(添加、編輯文字)
- ✅ 繪圖工具(畫筆、線條、箭頭)
- ✅ 形狀工具(矩形、圓形、多邊形)
- ✅ 高亮標記工具
- ✅ 註解與評論功能
- ✅ 圖片插入功能
- ✅ 頁面管理(旋轉、刪除、重新排序)
- ✅ 撤銷/重做支援

### 🔄 格式轉換
- PDF ↔ 圖片 (PNG/JPG)
- 圖片 → PDF
- PDF 合併
- PDF 分割

### 💻 雙平台支援
- **網頁版**: 可在瀏覽器直接使用
- **桌面版**: Windows 應用程式 (開發中)

## 🚀 快速開始

### 網頁版

1. **安裝依賴**
```bash
cd pdf-editor-web
npm install
```

2. **啟動開發伺服器**
```bash
npm run dev
```

3. **開啟瀏覽器**
訪問 http://localhost:5173

4. **建構生產版本**
```bash
npm run build
```

## 📖 使用說明

### 基本操作

1. **開啟 PDF 檔案**
   - 點擊工具列的「開啟」按鈕
   - 選擇要編輯的 PDF 檔案

2. **頁面導航**
   - 使用側邊欄的縮圖預覽切換頁面
   - 點擊縮圖即可跳轉到該頁面

3. **縮放控制**
   - 使用工具列的放大/縮小按鈕
   - 點擊「符合頁面」按鈕自動調整大小

4. **選擇編輯工具**
   - 點擊工具列的工具圖標
   - 可用工具:選取、文字、繪圖、形狀、高亮、註解、圖片、橡皮擦

5. **儲存編輯**
   - 點擊「儲存」按鈕
   - 編輯後的 PDF 會自動下載

### 格式轉換

1. **開啟轉換面板**
   - 點擊工具列的「轉換」按鈕(開發中)
   - 或使用快捷鍵

2. **選擇轉換類型**
   - PDF 轉圖片
   - 圖片轉 PDF
   - PDF 合併
   - PDF 分割

3. **上傳檔案**
   - 拖放檔案到上傳區域
   - 或點擊「瀏覽檔案」按鈕

4. **開始轉換**
   - 點擊「轉換」按鈕
   - 轉換完成後自動下載

### 設定

- **語言切換**: 點擊工具列的地球圖標
- **主題切換**: 點擊工具列的太陽/月亮圖標
- **側邊欄**: 點擊側邊欄圖標顯示/隱藏頁面縮圖

## 🛠️ 技術架構

### 前端技術棧
- **框架**: React 18 + TypeScript
- **建構工具**: Vite
- **PDF 處理**: 
  - PDF.js (渲染)
  - pdf-lib (編輯與生成)
- **狀態管理**: Zustand
- **多語言**: i18next
- **UI 組件**: 自訂 CSS + Lucide React (圖標)
- **動畫**: Framer Motion

### 專案結構
```
pdf-editor-web/
├── src/
│   ├── components/          # React 組件
│   │   ├── Editor/         # 編輯器組件
│   │   ├── Tools/          # 工具組件
│   │   ├── UI/             # UI 組件
│   │   └── Converter/      # 轉換組件
│   ├── lib/                # 核心庫
│   │   ├── pdf-renderer.ts # PDF 渲染
│   │   ├── pdf-editor.ts   # PDF 編輯
│   │   ├── pdf-converter.ts# 格式轉換
│   │   └── i18n.ts         # 多語言配置
│   ├── locales/            # 翻譯檔案
│   ├── styles/             # 樣式檔案
│   ├── store/              # 狀態管理
│   └── App.tsx             # 主應用
├── public/
└── package.json
```
